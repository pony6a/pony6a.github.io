. 用户交互input

why:	网页上，app 输入账号与密码。

what： 用户交互input

how：

```python
# input: 出来的全部都是字符串类型。
username = input('请输入用户名：')
password = input('请输入密码：')
print(username,type(username))
print(password,type(password))
```

why:  生活中选择，回家，n条路，你走那条路，取决于心情。

what: if。

how：

+ 基本结构：

+ ```
  if 条件:
      结果
      
  # c: if{条件}{结果}
  ```

   ```python
   print(111)
